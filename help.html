To Create a Note:
  1. Press the add_circle_outline icon
  2. Type the new note title and text in the textfields
  3. Set a reminder (optional)
  4. Press the Add Note button

To Delete a Note:
  1. Press the delete icon
  2. Press the Delete Button on the popup that appears

To Edit a Note:
  1. Click on the note you want to edit
  2. Edit the fields you want to change
  3. Press the Save button
